# TE-3: Currency exchange-rate equilibration

*Thought experiment, part of the [PromiseGrid Wire Lab](../../harness-spec.md). This file is content-addressable; its hash is its pCID.*

Three agents issue tokens. Initial trade is naive. After 1000 simulated transactions, do prices stabilize? After 100,000? Now break one agent (default on 5% of redemptions). Does its currency drop appropriately? Outcome: a price-discovery curve and a default-detection latency.
