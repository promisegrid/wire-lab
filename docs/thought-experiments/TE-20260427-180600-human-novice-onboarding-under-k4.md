# TE-7: Human-novice onboarding under K4

*Thought experiment, part of the [PromiseGrid Wire Lab](../../harness-spec.md). This file is content-addressable; its hash is its pCID.*

Drop a `user-novice` stochastic agent into a K4 (no-kernel) community. Tally messages until they make a successful first transaction. Compare to the same drop into K1. Outcome: a user-experience cost of the various ingress models.
